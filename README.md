# Aditya Kumar Portfolio Website

A professional, responsive portfolio website for Aditya Kumar, a B.Tech (EC) fresher and entry-level candidate.

## Files

- `index.html` - Main portfolio page
- `css/style.css` - Complete responsive styling
- `js/main.js` - Mobile menu, active navigation, contact form and animations
- `assets/aditya-kumar-profile-optimized.jpg` - Hero image
- `assets/aditya-kumar-avatar.jpg` - About-section avatar image
- `assets/aditya-kumar-cv.pdf` - Provided CV file kept for optional future use

## How to Use

1. Open `index.html` in any browser.
2. Edit text in `index.html` if you want to update details.
3. Replace images inside the `assets` folder if a new professional photo is available.
4. Upload the folder to GitHub Pages, Netlify, Vercel, or any web hosting platform.

## Privacy Note

The website intentionally uses only portfolio-friendly details. Full address, date of birth, and marital status are not shown on the page because they are not recommended for a public portfolio website.
